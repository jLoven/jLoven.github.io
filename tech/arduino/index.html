Breathalyzer

The motivation for this project was to put people in control of their safety: ideally, the user clips the Arduino onto his car’s sun visor, so it is nearby when he enters his car. It would prompt him to breathe into it when (for example) it’s late at night and displays a different LED color depending on his blood-alcohol content. Plus, this device is a lot cheaper than breathalyzers for sale today.  

 

 

 

As is, the user presses a button on the side of the device case to activate the ethanol sensor. The sensor takes a minute to heat up, and while it does so it cycles through the LED colors: red, blue, and green. Ideally, it would not cycle because it tells the user they are varying levels of drunk before settling, so I would like to be able to heat up the sensor in a more timely manner.  

 

 

 

We calibrated the sensor with alcohol. We researched legal limits for alcohol and assigned sensor colors to those limits (red means a BAC for which it is illegal to drive, blue means a BAC for which it is legal to drive, and green means no alcohol was detected).  

 

 

 

In the future, I would like to add voice to the device to tell the user whether they should call a cab or drive home. Check it out on Github here.

Arduino Rickroller

Another Arduino project I worked on was a device to Rickroll entrants to a room.

It uses an infrared motion sensor to detect movement, and when it detects movement, it uses a hardcoded sequence of tones and note durations to play a decent approximation of Rick Astley’s famous musical piece.

I worked on this project with Owen Lynch. He made the tonal approximations and we worked together to set up the hardware.

We have yet to find a worthy permanent location for the Arduino Rickroll device.

Check it out on Github here.
